## Context

The bachelor project provided the foundation for this thesis and should be seen as complementary work to it. Most of the information in the project report is now present in the README files throughout the implementation, but the report is kept here for completeness.

Because this thesis builds directly on the project implementation, some parts are similar or changed only slightly, while others have been changed significantly or implemented entirely from scratch.

## Note on relevance

Most of the content in this report is **outdated** with respect to the current implementation. However, parts of it remain relevant and can still be referenced, particularly:

- How to create new behavior/condition modules
- How to add new sensor data to the reference model
- How to set up a virtual machine (UTM) and install all dependencies

These topics are also described in the repository README files, but the project report provides additional background and detail.
